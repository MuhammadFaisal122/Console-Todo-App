# Console Todo App

A simple console-based todo application built in Python.

## Features

- Add new todo items
- View all todo items
- Update todo item descriptions
- Delete todo items
- Mark todo items as complete/incomplete

## Requirements

- Python 3.13 or higher

## Usage

Run the application with:

```bash
python -m src.todo_app.main
```

## Project Structure

- `src/todo_app/` - Main application source code
- `tests/` - Unit and integration tests
